# Проект информационной системы управления для электромонтажного ИП

## О проекте
Разработка конфигурации на платформе 1С:Предприятие 8.3 (учебная версия) для автоматизации учёта заказов, материалов, работников и клиентов электромонтажного предприятия. Созданы разграничения доступа для сотрудников. Проект выполнен по полному циклу: от обследования до создания работающего прототипа.

## Стек и инструменты
- 1С:Предприятие 8.3 (учебная версия)
- Проектирование: UML (Use Case, ER-диаграммы)

## Этапы проекта
### 1. Обследование и анализ
Проведён анализ бизнес-процессов ИП. Выявлены проблемы:
- Учёт заявок вёлся в Excel, без истории изменений
- Сложности с расчётом стоимости работ и материалов
- Отсутствие единой базы клиентов

### 2. Проектирование
Разработаны:
- Техническое задание (см. [ТЗ.pdf](./docs/TZ.pdf))
- Use Case диаграмма ролей пользователей
- ER-диаграмма базы данных (сущности: Заказ, Клиент, Материал, Бригада)
- Схема документооборота

### 3. Реализация в 1С
Создана конфигурация, включающая:
- **Справочники:** Номенклатура, Контрагенты, Сотрудники
- **Документы:** Заказ-наряд, Счёт на оплату, Приход материалов
- **Отчёт:** Расход материалов по объектам
- **Роли:** Администратор, Менеджер, Мастер
